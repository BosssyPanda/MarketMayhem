Market Mayhem: Java Trading Academy

Codex/Claude builds the game. You learn Java by playing it — no source editing, no passive notes. Every bit of Java XP is earned by answering an active challenge correctly.

How it works

You run one Java file in your terminal and start trading. Grow $1000 into $1500+ across 15 market days. To unlock the trading tools that give you an edge, you master Java topics inside the Java Trading Academy — one correct answer at a time.

  • Build the game by playing it. There is no StudentWork.java and no fill-in-the-blank source files.
  • No XP for reading. Opening a menu or reading a hint earns nothing.
  • All XP requires a correct answer. Master a topic with 3 correct answers to unlock its trading tool — permanently.

For Loops While Loops Arrays Methods Sequential Search Binary Search

Run it locally

Tap a command to copy it, then paste it into your terminal to play.

# In the project folder — compile & play 
javac MarketMayhem.java && java MarketMayhem

Don't have the files yet? Copy this to clone, build, and play in one go (it skips the clone if the folder already exists):

# Fresh machine — clone, build & play 
git clone https://github.com/BosssyPanda/MarketMayhem.git 2>/dev/null; cd MarketMayhem && javac MarketMayhem.java && java MarketMayhem

Beginner Java only — no ArrayList, HashMap, streams, lambdas, sorting libraries, or java.util.Random. One file, one Scanner, integers parsed by hand.
