Market Mayhem

A terminal-based Java stock trading game built to teach AP-style Java through arrays, loops, methods, input validation, and clean game flow.

View Source 
MARKET MAYHEM
=============
Day: 1 / 15
Cash: $1000
Portfolio: $0
Net Worth: $1000

0. Exit
1. View Market
2. Advance Day
3. View Portfolio
4. Buy Stock
5. Sell Stock

Run Locally

javac MarketMayhem.java
java MarketMayhem

v0.1 Complete

Includes market display, random daily price changes, portfolio tracking, buy/sell trades, and final win/loss summary.

Beginner Java

Uses one Java file, static methods, arrays, indexed loops, Scanner, and terminal input/output only.
